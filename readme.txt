WordPress 文章点赞插件


== Installation ==

1. 上传 `wp-postlike`目录 到 `/wp-content/plugins/` 目录
2. 在后台插件菜单激活该插件
3. 添加 `<?php wp_postlike();?>` 到需要的位置
